# CPU(Central Processing Unit), 중앙 처리 장치

**멀티 코어**

- 한 개의 칩 안에 여러 개의 코어가 있는 것

| 코어의 개수 | 명칭        |
| ----------- | ----------- |
| 1           | 싱글 코어   |
| 2           | 듀얼 코어   |
| 3           | 트리플 코어 |
| 4           | 쿼드 코어   |
| 6           | 헥사 코어   |
| 8           | 옥타 코어   |

**CPU 제조사**

- **인텔(Intel)** - 코어, 제온, 펜티엄
- **AMD** - 라이젠(RYZEN), 스레드리퍼, EPYC
- **애플(Apple)** - M1
- **ARM** - Cortex

**CPU**

- 코어가 많으면 더 많은 태스크(Task)를 빠르게 처리할 수 있다.
  - 태스크(Task) : 게임, 포토샵, 동영상 ...
- **스레드(Thread)** : CPU가 업무를 수행하는 단위
  - 한 개의 코어 당 하나의 스레드를 가지는 것이 원칙

- **캐시(Cache)** 메모리
  - 속도가 빠른 CPU 코어와 CPU에 비해 속도가 느린 메모리 사이에 속도 차에 따른 병목 현상을 완화하는 역할 


**멀티스레딩**

- 인텔의 경우 하이퍼스레딩
- AMD의 경우 SMT라고 부름
- 멀티 코어 프로세싱은 멀티 코어가 복수의 프로그램을 동시에 구동하는 데에 효과적
- 멀티스레딩 방식은 하나의 프로그램을 구동하는데 있어 전산 처리를 복수의 경로로 진행하기 때문에 하나의 코어가 가진 성능을 최대한 효율적으로 이용할 수 있게 도와줌

**하이퍼 스레딩(Hyper-Threading Technology)**

**CPU 쿨러**

- CPU는 연산처리를 많이하기 때문에 열이 발생한다. 그렇기 때문에 열을 식히기 위해 쿨러가 필요하다.
  - 효율적인 열 배출은 고장과 화재를 방지하고 컴퓨터의 성능에도 영향을 끼친다.
- 공랭방식, 수랭방식이 있다.

# 메인 보드

- Wi-Fi를 지원하는 메인 보드도 있다.
- 인텔은 세대별로 이름이 있다. (ex. 6세대 - 스카이레이크, 7세대 - 카비레이크, 8세대 - 커피레이크 ...)
- 인텔의 경우 CPU만 있어도 GPU 연산이 가능하기 때문에 모니터 출력이 가능하다.
- AMD의 경우 모델에 따라 CPU가 GPU 연산을 못하기 때문에 따로 GPU를 꽂아줘야 한다.
- 메인보드에 GPU, 전원 포트, RAM, CPU, USB, 랜선, SSD(HDD) 등이 연결된다.

# RAM(Random Access Memory), 주기억장치

- SSD(HDD) : 파일, 동영상 파일 등이 저장되어 있는 곳, 저장장치
- 동영상을 플레이하게 되면 저장장치에서 RAM으로 가져와서 CPU가 RAM에 있는 것을 처리하게 된다.
- CPU에서 저장장치까지 가서 처리해야될 일을 가져오려고하면 시간이 너무 오래 걸려서 RAM에 담아두고 일 처리를 한다.
- 휘발성, 전원을 끄게되면 다 날아간다.
- 하드한 작업을 하는게 아니면 8 ~ 16GB 정도면 충분하다.
- DDR3(예전 컴터용), DDR4
  - 데스크탑과 노트북은 다른 메모리를 사용한다.
- RAM도 뜨거워 지기 때문에 방열판을 붙여주면 성능 향상에 도움이 된다.
  - 하지만 그렇게 다이나믹한 효과를 주지는 않는다.

# HDD(Hard Disk Drive), SSD(Solid-State Drive), 보조기억장치

- 보조기억장치에 저장된 데이터를 주기억장치로 불러와서 주기억장치에서 CPU로 불러와서 일을 처리하게 된다.
- HDD 하드디스크는 물리적으로 쓰는 하드디스크
  - 원판을 회전시켜서 레이저로 쏴서 각인시켜서 동작함 
  - 컴퓨터 부품 중에서 유일하게 물리적인 방식으로 구현되는것
- RPM(분당 회전수) 원판이 회전하는 회전 수 높을 수록 가격과 속도가 빠르지만 소음이 있을 수 있다.
- SSD는 250GB or 500GB로 운영체제를 설치하는 용도로 사용 
  - HDD와 가격 차이가 많이 난다. 
- 동영상 같은거는 HDD에 

# Power

- W(와트)가 높을 수록 가격이 비싸다.
- 보통 500~600W로 컴퓨터에서 돌아가는 전력을 모두 공급할 수 있다. 하지만 그래픽 카드를 사용하게 될 경우는 다르다.
  - 그래픽 카드가 전력을 많이 잡아 먹기 때문에 고사양의 게임이나 프로그램을 돌릴경우에는 전력을 700W 이상의 것을 사용해야 한다.

# GPU(Graphics Processing Unit), 그래픽 처리 장치

- 그래픽 처리 장치로 CPU와 동일한 아키텍쳐를 가지고 있으면서 그래픽 관련 기본 연산을 처리할 수 있는 명령어를 담고 있어서 CPU보다 더 빠르게 그림을 그릴 수 있는 장치
- CPU는 하나하나 점을 찍어서 화면을 출력하지만 GPU는 한번에 병렬적으로 점을 찍어낸다.
  - 그 만큼 빠르지만 전력소비도 크고 비싸다.
  - 그래픽 칩 제조사는 인텔, NVIDA(GTX, RTX시리즈), AMD(RX 시리즈)
- 1000번대 미만의 그래픽 카드는 사무용에 적합

# 모니터

**크기**

데스크탑용 모니터

- 21인치, 23인치, 27인치, 32인치가 있다.

랩탑 용 모니터

- 13.3인치, 14인치, 15.6인치, 16인치, 17인치가 있다.

모니터 대각선 꼭짓점과 꼭지점 사이의 길이를 기준으로 한 것이다.

개인적으로 32인치 보다 27인치 듀얼로 사용하는게 좋다.

**해상도**

- 1920 X 1080 (FHD)
- 2560 X 1440 (WQHD)
- 3840 X 2160 (UHD/4K)
- 5120 X 2880 (UHD+/5K)

숫자는 픽셀의 갯수를 의미한다.

1인치 당 픽셀의 밀도를 나타내는 단위를 PPI(Pixels per inch)라 하며, 애플의 경우 PPI가 326을 넘어가게 되면 인간의 눈으로 화소를 구분할 수 없다고 하여 레티나 디스플레이라는 브랜드 이름을 붙여주기도 한다.

**주사율 (헤르츠)**

60Hz, 120Hz, 144Hz 등이 있다.

모니터가 표현할 수 있는 초당 프레임의 갯수를 나타낸다.

수치가 높을 수록 부드러운 화면을 표현하기 때문에 순간 순간이 중요한 게임의 경우 주사율이 높은 모니터가 선호된다.

**명암비**

완전한 검은색부터 완전한 흰색까지 표현하는 단계를 나타내는 말

단계가 많으면 많을 수록 더 선명한 화면을 볼 수 있다.

높은 명암비를 가진 모니터 HDR(High Dynamic Range) 디스플레이 기기로 인증 받을 수 있으며 HDR10의 경우 명암비가 1,080,000:1 이상이라고 볼 수 있다.

**패널 종류**

- TN(좁은 시야각, 역사가 오래된 패널), IPS, PLS 등의 다양한 패널 존재
- 액정의 배열 방식을 의미

**LCD & LED**

LCD : 액정 모니터, 전기로 액체의 방향을 조절하여 이미지를 구성

장점 : 가격이 저렴, 내구성이 좋음

단점 : 주사율과 명암비가 떨어지며 전력 소모가 심함

LED : 빛을 내는 반도체가 모여 이미지를 구성

장점 : 높은 주사율과 색표현력, 전력소모가 적음

단점 : 가격이 비싸고 내구성이 떨어진다.

피벗 모니터 

모니터를 돌릴 수 있는 것, 서브 모니터로 많이 사용한다.

DELL이 유명하고 LG나 삼성도 유명하다

모니터를 고를 때 디자이너 or 프론트엔드 개발자는 좋은 모니터를 사용하는게 좋다 (색감 표현 때문에)

# 키보드

총 3가지로 나눌 수 있음 멤브레인 키보드, 펜타그래프 키보드, 기계식 키보드

**멤브레인 키보드**

- 가장 대중적인 키보드, 보통 사무용으로 사용되며 대략생산으로 가격이 저렴하다.
- 내부에 있는 두장의 필름으로 전기가 통하면 키가 눌러지는 형태
- 장점 : 부드러운 키감으로 장시간 사용해도 피로감이 적음
- 단점 : 동시입력이 어려워 게임용으로 사용하기엔 무리가 있음

**펜타그래프 키보드**

- 주로 노트북에서 사용되는 키보드로 매우 슬림하며 적은 힘으로 장시간 사용해도 피로도가 낮다.
- 애플에서는 팬터그래프 키보드보다 더 얇은 노트북용 키보드(나비식 키보드)를 제작함
- 가위식, 나비식
- 가위식 버튼은 x자 지지대를 통해 키를 인식하는 형태이며 나비식 키보드는 돔 스위치를 통해 키를 인식함
- 나비식 키보드는 스트로크(키보드를 눌렀을 때 버튼이 얼만큼 내려갔는지)가 매우 짧아 얇은 노트북을 만들 수 있으나 키를 누르는 맛이사라짐.
- 터치식과 비슷한 감

**기계식 키보드**

스프링의 반발력을 이용하여 내구성을 높이고 타감을 높인 키보드

모든 키를 동시에 입력이 가능하며 다양한 부품들이 들어가 있어 다른 키보드들 보다 가격이 비싼 편이다.

청축, 갈축, 등이 있다. 겉으로 보기에는 어떤 키인지 구분할 수 없어 키캡을 뽑았을 때 나오는 스위치부의 색으로 구분할 수 있다.

해피해킹 키감이 좋아서 많이 사용함, 방향키가 없어서 FN키로 방향키를 조정해야됨
